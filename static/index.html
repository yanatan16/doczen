Functions (are awesome)

Functions are mini-programs that you give a name to. Then, you can run them again & again, just by calling their name.

Let's take a peek…

Creating & Calling a Function

Let's make a function called shinerdog that prints out "woooof!" every time we call it.

function shinerdog() {
    console.log("WOOOOF!")
}
